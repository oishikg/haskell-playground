# Haskell Playground

## Context
Repository documenting experiments with/explorations of different Haskell features.

## Structure
The project is set up as a stack project. Each module in `src` explores a different topic. 

## To run
Required: 
- GHC 9.2.5
- Stack 

Use the `ide` or `repl` commands in the Makefile to play around with the experiments/explorations.
